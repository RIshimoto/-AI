## 概要
卒業論文の際に制作したAlphaZero[^1]アルゴリズムを用いた京都将棋[^2]のゲームAIです。</br>

## デモ

## インストール
```
$ git clone https://github.com/RIshimoto/KyotoshogiAI
$ cd KyotoshogiAI
$ conda create -n kyotoshogi python=3.9
$ conda activate kyotoshogi
(kyotoshogi)$ pip install .
```
### プレイ
```
(kyotoshogi)$ play
```

### 学習
```
(kyotoshogi)$ train
```
